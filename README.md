Web mình sẽ bao gồm 3 phần: frontend + backend + database
Frontend được viết bằng js và css (react)
Backend được viết bằng node.js (express)
Database được viết bằng .sql
Chạy local: Kết nối giữa vscode với mysql workbench câu lệnh 
cd \quiz-management\backend node server.js để kết nối với backend và database
cd \quiz-management\ npm start ( để chạy giả lập react trên địa chỉ local)
Deploy trên web: 
Frontend sẽ deploy bằng vercel
Backend sẽ deploy bằng render
Database sẽ được chuyển sang TablePlus để deploy lên railway
Các phần sẽ được liên kết thông qua các file .env trong frontend và backend, và các biến môi trường (environment variables)
